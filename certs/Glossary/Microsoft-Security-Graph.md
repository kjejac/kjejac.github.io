(Microsoft Graph Security API)

Microsoft Security Graph er et felles sikkerhetslag som samler data fra flere Microsoft‑sikkerhetsprodukter og gjør dem tilgjengelige gjennom en API. Det gir en samlet måte å hente, korrelere og automatisere sikkerhetsdata på tvers av Microsoft‑tjenester.

- En API for alle Microsoft‑sikkerhetskilder
- Samler og korrelerer varsler og hendelser fra flere sikkerhetsprodukter
- Forenkler automatisering, analyse og respons på trusler
