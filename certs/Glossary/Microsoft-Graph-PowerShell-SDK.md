Microsoft Graph PowerShell SDK er en samling av moduler som lar deg administrere Microsoft Graph ressurser direkte med PowerShell kommandoer.
- kan installeres med PowerShellGet
- SDKen fungerer som et API-omslag for Microsoft Graph APIene og eksponerer hele API settet for bruk i PowerShell
