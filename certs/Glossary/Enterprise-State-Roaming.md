Enterprise State Roaming er en funksjon som synkroniserer brukerinnstillinger og appdata mellom enheter. Den giren sømløs og konsistent brukeropplevelse.

ESR har tett integrasjon med Entra ID. Innstillinger følger brukeren ved enhetsbytte eller reprovisjonering.

ESR forenkler klargjøring og administrasjon av enheter, og erstatter behovet for tradisjonelle roaming‑profiler. Den sikrer konsistente innstillinger på tvers av Windows‑enheter. 

Krever Microsoft Entra ID P1, P2 eller EMS‑lisens.

[Enable Enterprise State Roaming in Microsoft Entra ID](https://learn.microsoft.com/en-us/entra/identity/devices/enterprise-state-roaming-enable)
