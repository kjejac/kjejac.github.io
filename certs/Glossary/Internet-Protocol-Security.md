_IPsec (Internet Protocol Security)_ er en samling protokoller som sikrer kommunikasjon på IP‑nivå gjennom _kryptering, autentisering og integritetskontroll_. Det brukes både for VPN‑forbindelser og for å beskytte trafikk mellom enheter og nettverk.

- En _protokollsuite_ som beskytter IP‑trafikk (Cisco, Cloudflare).
- Gir _kryptering_, _autentisering_, _integritet_ og _anti‑replay‑beskyttelse_.
- Brukes i _VPN‑løsninger_, spesielt _L2TP/IPsec_ og _IKEv2/IPsec_ (Microsoft Docs).
- Kan sikre trafikk _maskin ↔ maskin_, _nettverk ↔ nettverk_, eller _klient ↔ nettverk_ (Palo Alto Networks).
