Microsoft Intune er et skybasert verktøy for en helhetlig enhetshåndtering, app-styring og sikkerhet. 
Støtter både BYOD- og bedriftseide-enheter. 

### Funksjoner
- Administrasjon av enheter, støttede operativsystemer:
	- Windows
	- macOS
	- iOS
	- Android
	- Linux
- Sikring av tilganger til bedriftsdata ved hjelp policyer og regler
- Distribusjon og oppdatering apper på tvers av enheter
- Håndtering av konfigurasjoner, for eksempel WiFi, VPN og sertifikater
- Beskytelse av data gjennom compliance-regler, kryptering og betinget tilgang
- En sentral portal for å overvåke, feilsøke og administrere alt fra skyen
### Bruk
- Kan bruks for hybrid og remote arbeid, der ansatte jobber fra mange steder og enheter
- Reduserer behovet for lokal infrastruktur
- Tett integrert med Microsoft 365, Entra ID og sikkerhetsverktøy
