Microsoft Copilot for Security er en generativ AI-drevet sikkerhetsløsning som hjelper sikkerhetsteam med å analysere, forstå og respondere på trusler raskere.

Løsningen genererer svar på brukerforespørsler ved hjelp av plugins som henter inn organisasjonsspesifikk informasjon, autoritative kilder og global trusselintelligens. 
Ved å bruke plugins som datakilder får sikkerhetsanalytikere bedre innsikt og mer kontekst, og hendelseshåndtere kan utvide funksjonaliteten ytterlige gjennom egne plugins.

Security Copilot integrerer med blant annet [Microsoft Intune](Microsoft-Intune.md) og [Microsoft Entra ID](Microsoft-Entra-ID.md), noe som gjør den relevant i moderne endepunkt- og identitetsadministrasjon.
