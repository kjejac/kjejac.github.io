Windows 365 er en skybasert Software-as-a-Service (SaaS) tjeneste som leverer en personlig, og vedvarende Windowsmaskin direkte fra Azure. Tjenesten oppretter automatisk en virtuelle Windows PC for hver bruker, og gir  tilgang til Windows 10/11 fra ulike enheter gjennom en nettleser eller RDP-klient.

### Kjernefunksjoner
- _Personlig Cloud PC_, hver bruker får sin egen dedikerte, vedvarende Windows-maskin i skyen
- _Windows 10/11 fra alle enheter_, tilgang fra alle enheter med en nettleser eller RDP-klient
- _Fast ytelse_, maskinen har faste ressurser som ikke deles med andre 
- _Enkel administrasjon_, PC administreres via Intune på samme måte som fysiske PCer
- _Sømløs brukeropplevelse_, brukeren kan fortsette arbeidet på tvers av enheter uten avbrudd

### Integrasjon med Intune
- _Provisioning via Intune_, PCer opprettes og tildeles automatisk basert på provisioning policyer
- _Enhetsadministrasjon_, PCer administreres som fysiske klienter gjennom policyer, apper og sikkerhet
- _Entra ID integrasjon_, pålogging og tilgang styres via Entra ID
- _Conditional Access_, tilgang til PC sikres med Conditional Access policyer
- _Sikkerhet_, onboarding til Defender for Endpoint og inngår i compliance policyer
