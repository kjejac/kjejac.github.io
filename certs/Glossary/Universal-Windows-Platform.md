Universal Windows Platform (UWP) er en app‑modell for Windows 10 og Windows 11 som lar utviklere bygge apper som kan kjøre på tvers av mange typer Windows‑enheter — PC, nettbrett, Xbox, HoloLens, Surface Hub og IoT‑enheter.

Hva UWP er
- En plattform som gir felles API‑overflate på alle Windows‑enheter.
- Apper kan bruke både UWP‑APIer, Win32‑APIer og .NET‑klasser.
- UWP‑apper er sandboxed, deklarerer hvilke ressurser de trenger, og er sikre å installere og avinstallere.

Viktige egenskaper
- Sikkerhet: Brukeren må godkjenne tilgang til kamera, mikrofon, filer osv.
- Tilpasning: UI kan automatisk tilpasse seg skjermstørrelse, DPI og ulike input‑metoder (touch, mus, tastatur, penn, gamepad).
- Distribusjon: Kan publiseres i Microsoft Store og installeres uten å påvirke systemet.
- Engasjement: Støtter live tiles, push‑varsler, bakgrunnsoppgaver og Cortana‑integrasjon.
- Fleksibilitet: Kan skrives i C#, C++, VB eller JavaScript, med XAML, HTML eller DirectX som UI‑lag.
