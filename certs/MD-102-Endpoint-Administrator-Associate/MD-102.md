# MD-102 Microsoft 365 Endpoint Administrator

MD-102 handler om administrasjon av enheter og applikasjoner i Microsoft 365-miljøer ved hjelp av Microsoft Intune. Sertifiseringen dekker:
- distribusjon og administrasjon av [endepunkter](Endpoint.md) 
- storskala administrasjon ved hjelp av: 
	- [Microsoft Intune](Microsoft-Intune.md)
	- [Microsoft Intune Suite](Microsoft-Intune-Suite.md)
	- [Windows Autopilot](Windows-Autopilot.md)
	- [Microsoft Copilot for Security](Microsoft-Copilot-for-Security.md)
	- [Microsoft Defender for Endpoint](Microsoft-Defender-for-Endpoint-(Defender-ATP).md)
	- [Microsoft Entra ID](Microsoft-Entra-ID.md)
	- [Azure Virtual Desktop](Azure-Virtual-Desktop.md)
	- [Windows 365](Windows-365.md)
- implementere identitet, sikkerhet og tilgangsstyring
- konfigurasjon av policyer, oppdateringer og applikasjoner for endepunkter
