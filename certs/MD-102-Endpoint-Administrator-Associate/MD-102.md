# MD-102 Microsoft 365 Endpoint Administrator

MD-102 er en Microsoft sertifisering som tar for seg administrasjon av enheter og applikasjoner i Microsoft 365-miljøer ved hjelp av Microsoft Intune. Den dekker:
- distribusjon og administrasjon av [endepunkter](../Glossary/Endpoint.md)
- storskala administrasjon ved hjelp av:
	- [Microsoft Intune](../Glossary/Microsoft-Intune.md)
	- [Microsoft Intune Suite](../Glossary/Microsoft-Intune-Suite.md)
	- [Windows Autopilot](../Glossary/Windows-Autopilot.md)
	- [Microsoft Copilot for Security](../Glossary/Microsoft-Copilot-for-Security.md)
	- [Microsoft Defender for Endpoint](../Glossary/Microsoft-Defender-for-Endpoint.md)
	- [Microsoft Entra ID](../Glossary/Microsoft-Entra-ID.md)
	- [Azure Virtual Desktop](../Glossary/Azure-Virtual-Desktop.md)
	- [Windows 365](../Glossary/Windows-365.md)
- implementere identitet, sikkerhet og tilgangsstyring
- konfigurasjon av policyer, oppdateringer og applikasjoner for endepunkter
